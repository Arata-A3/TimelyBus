# TimelyBus (徳島バス 時刻表 & 到着予測アプリ)

徳島バスの最新時刻表（PDF）をいつでも手軽に確認でき、さらに目的地までの到着残り時間をリアルタイムにカウントダウンできるアプリです。

## 主な機能

- **時刻表PDFの閲覧 (メイン)**: 
  - 徳島バス（バイパス・老門経由など）の公式時刻表PDFを、メニューからすぐに開けます。
  - スマートフォンに保存したり、オフラインでも開きやすい構成です。
- **バス到着カウントダウン**:
  - 全40箇所以上の停留所から選ぶだけで、次のバスが来るまでの時間を秒単位で表示。
  - ダイヤ（平日・土日祝）は自動で判別されます。
- **直感的な操作**:
  - 停留所名のキーワード検索や、乗車予定に合わせた時刻の微調整も簡単。
- **スマホアプリとして利用可能**:
  - ブラウザの「ホーム画面に追加」機能を使うことで、通常のアプリと同じ感覚で起動できます。

## 使い方 (スマートフォン)

1. **ページを開く**: GitHub Pages等で公開されたURLにアクセスします。
2. **ホーム画面に追加する**:
   - **iPhone (Safari)**: 画面下の[共有ボタン] ➔ [ホーム画面に追加]
   - **Android (Chrome)**: 右上の[︙] ➔ [ホーム画面に追加]
3. **アプリとして起動**: ホーム画面に作成されたアイコンから、いつでも時刻表を確認できます。
